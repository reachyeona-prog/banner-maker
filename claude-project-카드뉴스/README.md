# 카드 뉴스 카피라이터 — Claude 프로젝트 셋업

페이히어 인스타그램 카드 뉴스(1080×1350) 카피를 자동 생성하는 Claude 프로젝트입니다. 기사·블로그 **본문 텍스트**를 던지면, 표지 1장 + 내지 3~6장 분량의 카피를 도구 호환 JSON으로 돌려줍니다.

## 셋업 (5분)

1. **[claude.ai/projects](https://claude.ai/projects)** → **새 프로젝트 만들기**
2. 이름: "카드 뉴스 카피 생성기" (자유롭게)
3. **Instructions** 영역에 `00_프로젝트_지침.md` 내용을 통째로 복사·붙여넣기 (파일 첨부 X, 본문 텍스트만)
4. **Knowledge / Files** 영역에 아래 파일들 업로드:
   - `01_글쓰기_스타일.md`
   - `02_카드뉴스_구조.md`
   - `03_예시_모음.md`
   - `04_출력_스키마.md`
5. 모델은 **Sonnet 4.6** 또는 **Opus 4.7** 권장. 카피 톤·일관성에 민감한 작업이라 4.x 계열이 안정적.

> Web search는 사용하지 않습니다. URL fetch 안정성이 낮아 **본문 직접 붙여넣기**를 표준 흐름으로 잡았어요. 본문 복붙이 결과 정확도도 더 높습니다.

## 사용 방법

프로젝트 채팅창에 본문 텍스트를 통째로 붙여 넣기:

```
아래 글로 카드 뉴스 만들어줘.

[페이히어 블로그/기사 본문 전체 …]
```

응답은 JSON 한 덩어리. 그 JSON을 banner-maker 도구로 가져가 카피 필드에 채워 넣으면 됩니다.

URL만 던지면 Claude가 "본문을 직접 붙여넣어 주세요"로 응답합니다 (추측 카피 생성 방지).

## 업데이트

카피 톤이 만족스럽지 않으면:
- 새로운 좋은 예시는 `03_예시_모음.md`에 추가
- 새로 발견한 규칙은 `01_글쓰기_스타일.md`에 추가
- Knowledge 파일을 수정해서 다시 업로드 (덮어쓰기)

프롬프트 튜닝은 1회성이 아닌 점진적 개선이 자연스러워요.

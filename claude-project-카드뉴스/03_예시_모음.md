# 카드 뉴스 예시 모음 (Few-shot)

실제 페이히어 카드 뉴스의 톤·구조를 학습하기 위한 레퍼런스. 새 카피를 작성할 때 이 예시들의 호흡과 흐름을 모방하세요.

---

## 예시 1 — 벚꽃 상권 매출 분석 (NEWS)

**제목**: 벚꽃 상권 매출, 어디가 얼마나 올랐을까?
**원문 가정**: 페이히어 결제 데이터 분석 — 2024년 벚꽃 시즌(4/3~4/12) 서울 주요 벚꽃 명소 상권의 매출 변동

```json
{
  "title": "벚꽃 상권 매출 분석",
  "source": "https://payhere.in/blog/cherry-blossom-trends/",
  "cover": {
    "type": "E1",
    "pill": "NEWS",
    "main1": "벚꽃 상권 매출,",
    "main2": "어디가 얼마나 올랐을까?"
  },
  "interiors": [
    {
      "type": "E2",
      "main1": "벚꽃 나들이가 소비로 이어지며",
      "main2": "{서울 주요 벚꽃 상권 매출}이 올랐어요!"
    },
    {
      "type": "E2",
      "main1": "가장 큰 증가세를 보인 잠실 · 종로,",
      "main2": "{어떤 업종 매출}이 늘었을까요?"
    },
    {
      "type": "E2",
      "main1": "{잠실}은 일식·주점·양식이",
      "main2": "최대 {5.4배}까지 뛰었어요"
    },
    {
      "type": "E2",
      "main1": "{종로}는 선물·기념품 매출이",
      "main2": "전월 대비 {61%} 늘었어요"
    },
    {
      "type": "E2",
      "main1": "벚꽃이 진 뒤에도",
      "main2": "이 트렌드, 우리 매장에 적용해 보세요"
    }
  ]
}
```

**해설**:
- 표지: 콤마 호흡 + 의문문. "어디가" "얼마나" 두 가지 궁금증을 묶어 강한 후킹.
- 내지 1: 맥락 → 결론 흐름. `{서울 주요 벚꽃 상권 매출}` 한 덩어리로 강조.
- 내지 2: 전환 + 질문으로 다음 슬라이드 유도.
- 내지 3·4: 구체 데이터. 지역·업종·배수 명시.
- 내지 5: CTA. 광고체 없이 자연스럽게 마무리.

## 예시 2 — 고유가 피해지원금 가이드 (매장 운영 TIP)

**제목**: 고유가 피해지원금 핵심 요약.zip
**원문 가정**: 정부 고유가 피해지원금 정책 안내. 자영업자 대상 신청 자격·금액·기간 정리.

```json
{
  "title": "고유가 피해지원금 가이드",
  "source": "https://payhere.in/blog/oil-price-relief/",
  "cover": {
    "type": "E1",
    "pill": "매장 운영 TIP",
    "main1": "고유가 피해지원금",
    "main2": "핵심 요약.zip"
  },
  "interiors": [
    {
      "type": "E2",
      "main1": "신청 자격은",
      "main2": "{연 매출 3억 이하} 사장님이에요"
    },
    {
      "type": "E2",
      "main1": "최대 {300만원}까지 받을 수 있고",
      "main2": "신청 기간은 {6월 15일까지}예요"
    },
    {
      "type": "E2",
      "main1": "신청 방법은 {정부24} 또는",
      "main2": "{관할 시·군·구청} 방문이에요"
    },
    {
      "type": "E2",
      "main1": "기간 놓치면 끝!",
      "main2": "{체크리스트}는 페이히어 블로그에서"
    }
  ]
}
```

**해설**:
- 표지: `.zip` 압축 패턴. "핵심 요약"이라는 카피로 정보성 콘텐츠임을 명확히.
- 내지: 매 슬라이드 한 정보(자격 / 금액·기간 / 신청 방법 / CTA).
- 강조: 숫자(`{300만원}`, `{6월 15일까지}`)와 지명(`{정부24}`)에 집중.
- 마지막: 가벼운 후킹 + CTA. "끝!"의 느낌표는 절제된 1개.

## 예시 3 — 외국인 결제 트렌드 (트렌드)

**제목**: 외국인이 한국에서 가장 많이 결제한 업종은?

```json
{
  "title": "2024 외국인 결제 트렌드",
  "source": "https://payhere.in/blog/foreign-payment-2024/",
  "cover": {
    "type": "E1",
    "pill": "트렌드",
    "main1": "외국인이 한국에서",
    "main2": "가장 많이 결제한 업종은?"
  },
  "interiors": [
    {
      "type": "E2",
      "main1": "2024년 외국인 결제 데이터,",
      "main2": "{어떤 업종}이 1위였을까요?"
    },
    {
      "type": "E2",
      "main1": "1위는 {일식},",
      "main2": "{전년 대비 35.6%} 늘었어요"
    },
    {
      "type": "E2",
      "main1": "그 뒤를 잇는 건",
      "main2": "{카페}와 {기념품숍}이에요"
    },
    {
      "type": "E2",
      "main1": "K-콘텐츠가 이끈 변화,",
      "main2": "우리 매장 메뉴에 반영해 볼까요?"
    }
  ]
}
```

**해설**:
- 표지: 단순 질문형. "한국에서 / 가장 많이 / 결제한"으로 호흡이 자연스러움.
- 내지 1: 질문 강조로 다음 슬라이드 시선 유도.
- 내지 2·3: 1위 → 2·3위 데이터 노출. 비교 흐름.
- 내지 4: 시사점. "K-콘텐츠"라는 트렌드 키워드 + 사장님 시점 CTA.

## 예시 4 — 최저임금 인상 (정책 가이드)

**제목**: 2025년 최저임금, 우리 매장에 얼마 영향?

```json
{
  "title": "2025 최저임금 영향 분석",
  "source": "https://payhere.in/blog/min-wage-2025/",
  "cover": {
    "type": "E1",
    "pill": "정책 가이드",
    "main1": "2025년 최저임금,",
    "main2": "우리 매장에 얼마 영향?"
  },
  "interiors": [
    {
      "type": "E2",
      "main1": "2025년 최저임금은",
      "main2": "시간당 {10,030원}이에요"
    },
    {
      "type": "E2",
      "main1": "전년 대비 {1.7%} 인상,",
      "main2": "월급 환산 {209만 6,270원}이에요"
    },
    {
      "type": "E2",
      "main1": "주휴수당까지 더하면",
      "main2": "사실상 {시급 12,036원} 수준이에요"
    },
    {
      "type": "E2",
      "main1": "직원 1명 기준 월 인건비,",
      "main2": "전년보다 약 {3.5만원} 늘어요"
    },
    {
      "type": "E2",
      "main1": "정확한 시뮬레이션은",
      "main2": "{페이히어 블로그}에서 확인하세요"
    }
  ]
}
```

**해설**:
- 표지: 사장님 직접 질문형 ("우리 매장에 얼마 영향?"). 친근하고 즉시 관련성.
- 내지: 숫자가 거의 모든 슬라이드에 등장. 정책 가이드의 핵심.
- 강조: 모든 슬라이드의 `{}`가 **숫자/금액** — 일관된 강조 패턴.
- 마지막: 페이히어 블로그로 자연스러운 연결.

---

## 응용 — 좋은 카드 뉴스의 공통점

1. **표지가 짧고 강하다** — 한 줄 14자 이하. 질문 또는 압축.
2. **내지마다 숫자가 있다** — 데이터 카드 뉴스의 본질.
3. **강조는 절제** — 슬라이드당 1~2개 `{}`.
4. **흐름이 끊기지 않는다** — 슬라이드 → 슬라이드가 "그래서?"의 답.
5. **마지막이 가볍다** — 광고체 없이 자연스러운 CTA 또는 시사점.

새 카피를 쓸 때 이 다섯 가지를 체크하세요.
